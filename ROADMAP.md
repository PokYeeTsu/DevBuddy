# DevBuddy Roadmap

## Phase 1 — Core Foundation (MVP)
- [ ] Repo setup
- [ ] File indexer (paths, metadata)
- [ ] SQLite database backend
- [ ] Simple search (filename, substring)
- [ ] CLI for indexing/searching
- [ ] Basic unit tests

## Phase 2 — Structural Search Expansion
- [ ] Parse file contents
- [ ] Extract functions, classes
- [ ] Store textual snippets
- [ ] Extend search to functions & definitions

## Phase 3 — Semantic Search
- [ ] Integrate embeddings (OpenAI or local model)
- [ ] Vector search (pgvector or SQLite-based)
- [ ] Semantic query CLI/API

## Phase 4 — API Layer
- [ ] FastAPI service
- [ ] Endpoint for search
- [ ] Endpoint for indexing
- [ ] JSON output for integrations

## Phase 5 — UI Prototype
- [ ] Basic web UI
- [ ] Query input + results view
- [ ] File preview
- [ ] Summaries & explanations

## Phase 6 — Long-Term Features
- [ ] Knowledge graph model
- [ ] Multi-language tree-sitter parsing
- [ ] Multi-repo support
- [ ] CI integration
- [ ] Plugins (Jira, GitHub, Sentry)
- [ ] Self-hosted LLM support
